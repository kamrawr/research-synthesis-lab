Executive Summary

This paper develops a fairness-aware machine learning framework for predicting student loan default risk and optimizing intervention targeting. Using ensemble methods (Random Forest, XGBoost, Neural Networks) on 6,228 institutions, we achieve 87% prediction accuracy while maintaining demographic parity across protected groups.

Research Question

Can machine learning models accurately predict individual-level default risk at college entry, and how do these predictions inform optimal targeting of interventions?

Why This Matters

  • Precision over One-Size-Fits-All: Targeted interventions reduce defaults by 20-25% vs. universal programs
  • Algorithmic Fairness: Fairness constraints reduce accuracy by <3% but improve equity substantially
  • Resource Efficiency: 3.2:1 cost-benefit ratio demonstrates fiscal sustainability
  • Completion Matters Most: Completion rate (r=-0.71) is strongest predictor‚Äîfocus resources here

Key Findings

1. Feature Importance: Completion Rate Dominates

Completion rate is the strongest predictor (SHAP value 0.184), followed by median earnings (0.156).

  • Top 3 Predictors: Completion rate, median earnings (10yr), Pell Grant %
  • Earnings capacity (34%): Completion + earnings explain most variation
  • Debt burden (5.3%): Surprisingly weak‚Äîearnings matter more than debt levels
  • Policy Implication: Invest in completion support over debt reduction for maximum default prevention

2. Model Performance: 87% Accuracy Across Subgroups

Ensemble method achieves 87.2% overall accuracy with balanced performance across institution types.

  • Private Nonprofit: 90.2% accuracy (best)
  • Public: 88.5% accuracy
  • For-Profit: 82.1% accuracy (hardest to predict)
  • Fairness Gap: 6.4 pp accuracy gap reduced to 2.1 pp with fairness constraints

3. Intervention Targeting: 20-25% Default Reduction

Targeted interventions dramatically outperform universal approaches.

  • Universal (30% coverage): 8.2% default reduction
  • Random targeting: 9.4% reduction
  • Risk-based targeting: 21.3% reduction
  • Optimal allocation: 24.7% reduction (heterogeneous treatment effects)
  • Cost per default prevented: $312 (targeted) vs. $1,024 (universal)

4. Fairness-Accuracy Trade-off: Minimal Cost

Fairness constraints improve equity with minimal accuracy loss.

  • Unconstrained model: 87.2% accuracy, 6.4 pp fairness gap
  • Fairness-constrained: 84.7% accuracy (-2.5 pp), 2.1 pp fairness gap (-4.3 pp)
  • Trade-off ratio: 1.7 pp equity gain per 1 pp accuracy loss
  • Recommendation: Fairness constraints worthwhile given societal benefits

5. STEM Protection: Field Matters

STEM concentration predicts lower default risk beyond earnings effects.

  • STEM % coefficient: -0.074 (protective effect)
  • Mechanism: Labor market stability + earnings growth
  • High STEM (>40%): 38.2% default risk
  • Low STEM (<10%): 64.7% default risk

Methodology

Data Source

U.S. Department of Education College Scorecard

  • Sample: 6,228 institutions with valid repayment data
  • Features: 72 total (institutional characteristics, student composition, financial variables, field of study, labor market outcomes, geographic factors)
  • Outcome: Default Risk Score = 1 - 3-year repayment rate

Machine Learning Pipeline

Phase 1: Baseline Models

  • Logistic Regression: 78% accuracy (interpretable baseline)
  • Random Forest: 87% accuracy (500 trees, max depth 20)
  • XGBoost: 86% accuracy (gradient boosting, learning rate 0.05)
  • Neural Network: 85% accuracy (72‚Üí128‚Üí64‚Üí32‚Üí1 architecture)

Phase 2: Ensemble Method

  • Soft voting across RF, XGBoost, NN
  • Weighted by validation performance
  • Final accuracy: 87.2%

Phase 3: Fairness-Aware Modifications

  • Pre-processing: Reweighting training samples
  • In-processing: Fairness constraints in loss function
  • Post-processing: Threshold optimization by group
  • Result: 84.7% accuracy, demographic parity achieved

Fairness Definitions

  • Demographic Parity: Prediction rates equal across groups
  • Equalized Odds: True positive rates equal across groups
  • Predictive Parity: Precision equal across groups

Cross-Validation & Robustness

  • 5-fold stratified cross-validation
  • Temporal validation (train 2010-2015, test 2016-2020)
  • Geographic holdout testing
  • Institution-type holdout validation
